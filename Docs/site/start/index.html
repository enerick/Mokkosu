Mokkosuの始め方

現時点でドキュメント・サンプルがともに不足していますが、 なるべく早くこの状況を改善したいと考えています。

現時点でMokkosuを始めてくださる方は、 以下のことをするとよいと思います。

  1. 標準ライブラリのソースコードを読む
  2. Sampleディレクトリの ソースコードを読む。
  3. 言語マニュアルを読む。
